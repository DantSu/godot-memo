# GODOT Mémo

Godot est un moteur de jeu vidéo 2D / 3D cross-platform gratuit, libre et open-source. 
Godot permet donc de développer des logiciels ou jeux vidéos 
sur desktop (Linux, MacOS, Windows), 
mobile (Android, iOS), 
Web.
